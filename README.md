## Цели и задачи

* Создать лёгкое расширение для браузера, которое доступно пользователям без установки тяжёлого софта.
* Обеспечить интересное обучение через геймификацию и интеракстив.
* Сделать удобный интерфейс, простую установку и быстрый старт без необходимости сложной настройки.
* Помочь пользователям понять как работает интернет. Миру пока неизвестна магия!

---

## Функциональные возможности

* Интеграция с браузером (напр., Chrome / Firefox) — расширение, доступное сразу после установки.
* Хранение прогресса.
* Удобный пользовательский интерфейс, простой старт без лишних настроек.

---

## Почему этот проект полезен / мотивация

* Многие люди пользуются интернетом годами, и даже опытный пользователь может попасть на фишинговый сайт, не зная как работает интернет
* Расширение помагает пользователю задумываться над работой браузера, и пользоватся им задумываясь над своими действиями
* Человек, используюший мое расширение сможет лучше фильтровать информацию в интернете, поточу что оно предоставляет информацию о популярных ресурсах
